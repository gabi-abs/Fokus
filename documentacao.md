# COMANDOS DO REACT NATIVE
npx create-expo-app@3 fokus
-> criar um app react native com o nome fokus, na versão 3.


npm start 
-> Prepara o servidor para ser executado 

npm run reset-project
-> Reseta o projeto para ficar limpo
